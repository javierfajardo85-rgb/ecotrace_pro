## Elevator pitch

EcoTrace es la infraestructura de contabilidad de carbono auditable que permite a los e-commerce automatizar el cálculo de emisiones de sus envíos bajo el estándar ISO 14064.

## Problema → solución

1. Excel manual → API + registro inmutable por transacción.  
2. Greenwashing / desconfianza → transparencia con datos científicos (DEFRA / GHG Protocol) visibles en checkout.  
3. CSRD / Green Claims → motor “Audit-Ready” para inspecciones.
